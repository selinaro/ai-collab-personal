# Message drafts — first person singular, kind tone

When drafting messages in Selina's name (Webex, Jira, e-mail), write **"ich", not
"wir"** — Selina works alone on her design-system work; "wir" misrepresents the team
situation.

**Why:** corrected 2026-07-09 when a Webex draft to a customer contact (Karin,
Polypoint) used "wir" throughout.

**How to apply:**
- Default every drafted message to first person singular unless Selina explicitly
  speaks for a group.
- Related: route clarification questions to the responsible senior contact (e.g.
  Karin at Polypoint) rather than to junior developers — phrase developer-facing
  communication as finished, appreciative specs, not open questions (Denny incident,
  2026-07-09).
